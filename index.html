Overview

Bootstrap-Jamon is a set of Bootstrap-aware helpers for the Jamon templating engine.

This is not a general purpose HTML library. It only eliminates the most boilerplate-heavy parts of Bootstrap. In fact, it does not cover every single possibility, but only those where it can significantly reduce the amount of code. If there's something the library doesn't cater for, don't hesitate to drop down to raw HTML.

Bootstrap-Jamon is currently based on Bootstrap 2.3.2

View on Github

This documentation is itself written in Bootstrap-Jamon and is a good source of examples.

Aliases

The raw template paths can get quite long. Here are some recommended aliases you can copy into your template files or jamon.properties.

These aliases will be used throughout the documentation.

For template files: 
<%alias>
bs=/com/moandjiezana/bootstrapjamon;
base=bs//base;
s=bs//scaffolding;
n=bs//components/navigation;
f=base//forms;
b=bs//base/buttons;
</%alias>
For jamon.properties: 
org.jamon.alias.bs=/com/moandjiezana/bootstrapjamon
org.jamon.alias.base=/com/moandjiezana/bootstrapjamon/base
org.jamon.alias.s=/com/moandjiezana/bootstrapjamon/scaffolding
org.jamon.alias.n=/com/moandjiezana/bootstrapjamon/components/navigation
org.jamon.alias.f=/com/moandjiezana/bootstrapjamon/base/forms
org.jamon.alias.b=/com/moandjiezana/bootstrapjamon/base/buttons

Scaffolding

include

For use in the <head>, includes Bootstrap's CSS and JS.

Input: 
<& s//include; responsive=true; bodyPadding=60 &>
Output: 
<script type="text/javascript" src="js/bootstrap.js"></script>
<link rel="stylesheet" href="css/bootstrap.css">
<style>
    body {
        padding-top: 60px;
    }
</style>
<link rel="stylesheet" href="css/bootstrap-responsive.css">
<!--[if lt IE 9]>
    <script src="//html5shiv.googlecode/svn/trunk/html5.js"></script>
<![endif]-->
<meta name="viewport" content="width=device-width, initial-scale=1.0">
Attribute Default Description
bootstrapPath "" The root path to Bootstrap's assets. The css, js and img folders should be found here. The default will point to the current folder. Should not end with a slash.
fileName "bootstrap" The base file name used to construct the CSS and JS asset paths.
cssFileName fileName The file name to which .css will be appended.
jsFileName fileName The file name to which .js will be appended.
responsive false If true, the responsive CSS will be included, using cssFileName as the base name.
bodyPadding 0 If greater than 0, adds padding-top to the body. Set this value if you are using a fixed navbar.
ieShim true If true, includes HTML5 shim, which enables HTML5 elements in IE older than IE 9.
meta true If true, adds the recommended viewport meta tag.

Examples
Input: 
<&| s//row &>
    <&| s//span12 &>
        // content...
    </&>
</&>
<&| s//row; fluid=true &>
    <&| s//span4; offset=3 &>
        // content...
    </&>
    <&| s//span5; extraClasses="custom-class" &>
        // content...
    </&>
</&>

<div class="row">
    <div class="span12">
        // content...
    </div>
</div>
<div class="row-fluid">
    <div class="span4 offset3">
        // content...
    </div>
    <div class="span5 custom-class">
        // content...
    </div>
</div>

Buttons

a

Turns a link into a button.

Examples
Default 
<&| b//a; href="#" &>Default</&>
Primary 
<&|b//a; href="#"; style=ButtonStyles.primary &>Primary</&>
test 
<&|b//a; href="#"; icon=Icons.edit; &>edit</&>
mini 
<&|b//a; href="#"; size=ButtonSizes.mini &>mini</&>
span3 Arbitrary sizes can be used via the extraClasses attribute. 
<&|b//a; href="#"; label="Primary"; extraClasses="span3" &>span3</&>
Attribute Default Description
href   The value of the href attribute
style ButtonStyles.none The look to apply to the button. The enum names match Bootstrap's CSS classes.
size ButtonSizes.none The size of the button. The enum names match Bootstrap's CSS classes.
icon null If set to a value from the Icons enum, will add the corresponding icon.
iconColor IconColors.AUTO Forces the icon to be black or white. AUTO uses black icons for light button styles and white icons for dark button styles.
disabled false If true, adds the .disabled class.
extraClasses "" Arbitrary additional CSS classes to add
