Overview

Bootstrap-Jamon is a set of Bootstrap-aware helpers for the Jamon templating engine.

It is not a general purpose HTML library. It only eliminates the most boilerplate-heavy parts of Bootstrap. In fact, it does not cover every single possibility, but only those where it can significantly reduce the amount of code.

Bootstrap-Jamon is currently based on Bootstrap 2.0.1

View source on Github

This documentation is itself written in Bootstrap-Jamon and is a good source of examples.

Aliases

The raw template paths can get quite long. Here are some recommended aliases you can copy into your template files or jamon.properties.

These aliases will be used throughout the documentation.

For template files: 
<%alias>
bs=/com/moandjiezana/bootstrapjamon;
base=bs//base;
s=bs//scaffolding;
n=bs//components/navigation;
b=bs//base/buttons;
</%alias>
For jamon.properties: 
org.jamon.alias.bs=/com/moandjiezana/bootstrapjamon;
org.jamon.alias.base=bs//base;
org.jamon.alias.s=bs//scaffolding;
org.jamon.alias.n=bs//components/navigation;
org.jamon.alias.b=bs//base/buttons;

Scaffolding

include

For use in the <head>, includes Bootstrap's CSS and JS.

Attribute Default Description
bootstrapPath "" The root path to Bootstrap's assets. The css, js and img folders should be found here. The default will point to the current folder.
cssFileName "bootstrap" The file name to which .css will be appended.
jsFileName "bootstrap" The file name to which .js will be appended.
responsive false If true, bootstrap-responsive.css will be included.

row

Attribute Default Description
fluid false Uses row class when false, row-fluid when true.
extraClasses "" Arbitrary CSS classes.

span1 - span12

Convenience templates with pre-defined sizes.

Attribute Default Description
offset 0 The size of the offset class to use, if greater than 0.
extraClasses "" Arbitrary CSS classes.

span

Template for use only on grids with more than 12 columns.
Attribute Default Description
size 12 The size of the span class to use.
offset 0 The size of the offset class to use, if greater than 0.
extraClasses "" Arbitrary CSS classes.

Examples

Input: 
<&| s//row &>
    <&| s//span &>
        // content...
    </&>
</&>
<&| s//row; fluid=true &>
    <&| s//span4; offset=3 &>
        // content...
    </&>
    <&| s//span5; extraClasses="custom-class" &>
        // content...
    </&>
</&>
Output: 
<div class="row">
    <div class="span12">
        // content...
    </div>
</div>
<div class="row-fluid">
    <div class="span4 offset3">
        // content...
    </div>
    <div class="span5 custom-class">
        // content...
    </div>
</div>

Base

Buttons

a

The a template turns a link into a button.

Examples

Default 
<&|  b//a; href="#" &>Default</&>
Primary Use the ButtonStyles enum to apply a look to your button. The enum names match Bootstrap's CSS classes. 
<&| b//a; href="#"; style=ButtonStyle.primary &>Primary</&>
mini Use the ButtonSizes enum to change the size. The enum names match Bootstrap's CSS classes. 
<&| b//a; href="#"; size=ButtonSizes.mini &>mini</&>
span3 Arbitrary sizes can be used via the extraClasses attribute. 
<&| b//a; href="#"; label="Primary"; extraClasses="span3" &>span3</&>

icon

All icons are available from the Icons enum. 
<& base//icon; icon=Icons.map_marker &>
button 
<&| b//a; href="#" &><& base//icon; icon=Icons.edit &> button</&>

Components

Navigation

navbar

Attribute Default Description
fixedTop false If true, adds the .navbar-fixed-top class.

dropdown

The outer part of the dropdown menu. The body of the template becomes the body of the inner <ul>.
Attribute Default Description
label   The text of the dropdown's header.
jsAuto true If true, will be activated via JavaScript on page load.
href "#" The header's href attribute.

navLink

A list item that contains a link.
Attribute Default Description
href   The link's href attribute.
active false If true, adds the active class to the list item.

navbarToggle

The default menu toggle button for responsive nav bars.
Attribute Default Description
target ".nav-collapse" The value of the data-target attribute.
